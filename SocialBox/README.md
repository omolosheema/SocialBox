# Installation
```
sudo apt-get install git
git clone https://github.com/omolosheemajuliansparks/SocialBox.git
cd SocialBox
chmod +x SocialBox.sh
chmod +x install-sb.sh
./install-sb.sh
./SocialBox.sh
```
